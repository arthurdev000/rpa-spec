Glossary
========

.. glossary::

   MMU
      Memory Management Unit. Hardware component responsible for
      virtual-to-physical address translation.

   PTE
      Page Table Entry. A single entry in a page table describing
      the mapping and permissions for a page.

   TLB
      Translation Lookaside Buffer. A cache that stores recent
      virtual-to-physical address translations.

   VPN
      Virtual Page Number. The part of a virtual address that
      identifies the page.

   PPN
      Physical Page Number. The part of a physical address that
      identifies the physical page frame.

   Sv39
      RISC-V virtual memory scheme with 39-bit virtual addresses
      and 3-level page tables.

   Sv48
      RISC-V virtual memory scheme with 48-bit virtual addresses
      and 4-level page tables.
